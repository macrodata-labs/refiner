# CLI

The `macrodata` CLI authenticates, submits scripts, inspects jobs, follows logs,
reads metrics, and manages secrets.

| Page | Commands |
| --- | --- |
| [Auth and Run](auth-and-run.md) | `login`, `whoami`, `logout`, `run` |
| [Jobs, Logs, and Metrics](jobs-logs-and-metrics.md) | `jobs list/get/attach/logs/metrics/resource-metrics/cancel` |
| [Secrets](secrets.md) | `secrets list/set/remove` |
