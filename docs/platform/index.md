# Platform

Refiner can run locally, but Macrodata Cloud adds shared execution, job
tracking, logs, metrics, secrets, manifests, and cloud files.

| [Cloud Jobs and Files](cloud-jobs-and-files.md) | Job tracking and file staging from the user perspective. |
